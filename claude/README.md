# 在 Claude 里使用 FIRE 规划顾问

本目录是给**不用 Cursor、只用 Claude** 的用户准备的。把下面的说明放进 Claude，就能让 Claude 按 FIRE 顾问的流程和你对话。

---

## 方式一：自定义说明（推荐）

1. 打开 **Claude.ai** 或 **Claude 桌面版**，进入 **设置 → 自定义说明**（或「Project → Custom Instructions」）。
2. 在「自定义说明」里粘贴 **`FIRE规划顾问-系统说明.md`** 的全文。
3. 保存后，新建对话，说「开始规划」或「我想做 FIRE 规划」，Claude 就会按顾问流程带你走。

---

## 方式二：每条新对话时用

不想改全局设置的话，可以在**每次新开一个 FIRE 规划对话**时：

1. 第一条消息先粘贴 **`FIRE规划顾问-系统说明.md`** 的全文。
2. 紧接着发一句：「请按上面的角色和流程，现在开始帮我做 FIRE 规划。」
3. 之后按 Claude 的提问回答即可。

---

## 方式三：当知识库用（Claude 项目 / 其他支持上传文档的产品）

若你用的 Claude 产品支持「上传文档」「项目知识库」：

- 上传 **`FIRE规划顾问-系统说明.md`** 作为项目说明或知识库。
- 再上传 **`模板-画像与规划.md`**（可选），方便 Claude 按模板输出画像和规划。

然后在对话里说「开始 FIRE 规划」或「按知识库里的 FIRE 顾问流程来」。

---

## 方式四：Cowork + 本地文件夹（不用粘贴）

若你用 **Claude 桌面版** 的 **Cowork** 模式（Pro / Max / Team / Enterprise）：

1. 把本 **`claude/`** 目录放在你电脑的任意位置（或整个 `fire-planner-cursor-plugin` 仓库克隆到本地）。
2. 在 Cowork 里**添加/共享这个文件夹**（Cowork 支持「给 Claude 一个本地文件夹，它可读、可写该文件夹内的文件」）。
3. 新建任务，在对话里直接说：「请读这个文件夹里的 `FIRE规划顾问-系统说明.md`，按里面的角色和流程，现在开始帮我做 FIRE 规划。」

Claude 会从你共享的文件夹里**直接读文件**，无需复制粘贴；完成的画像和规划也可以让它写回同一文件夹。

---

## 文件说明

| 文件 | 用途 |
|------|------|
| `FIRE规划顾问-系统说明.md` | 完整角色 + 流程 + 输出格式，粘贴到自定义说明或第一条消息 |
| `模板-画像与规划.md` | 画像、规划的 Markdown 模板，可选上传当参考 |
