# 两步都做：GitHub 公开 + 提交官方 Marketplace

## 第一步：推到 GitHub（让人通过链接看见、安装）

1. 在 GitHub 新建一个**空仓库**（例如名字：`fire-planner-cursor-plugin` 或 `cursor-plugin-fire-planner`），不要勾选 README / .gitignore（避免冲突）。
2. 在本机进入插件目录并推送：

   ```bash
   cd fire-planner-cursor-plugin
   git init
   git add .
   git commit -m "Initial: FIRE 路径规划顾问 Cursor Plugin"
   git branch -M main
   git remote add origin https://github.com/<你的用户名>/<仓库名>.git
   git push -u origin main
   ```

3. 之后把仓库设为 **Public**，把链接发到知乎/即刻/社群等，别人即可按 README 安装。

---

## 第二步：提交 Cursor 官方 Marketplace（审核通过后所有人能在 Cursor 里搜到）

1. 确保上一步的仓库已是**公开**且内容已 push 完成。
2. 打开：**[https://cursor.com/marketplace/publish](https://cursor.com/marketplace/publish)**
3. 在页面里提交你的**仓库 URL**（例如 `https://github.com/用户名/fire-planner-cursor-plugin`）。
4. 等待 Cursor 团队审核，通过后插件会出现在 [cursor.com/marketplace](https://cursor.com/marketplace)，用户可在 Cursor 内直接搜索安装。

---

## 可选：发布前自检

- [ ] 仓库根目录有 `README.md`、`LICENSE`  
- [ ] 本地在 Cursor 里试过该 skill 能正常触发、使用
